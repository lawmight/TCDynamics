# Squad

Quick alias for `/all-agents` - launches all 4 specialized subagents in parallel with coordinated execution.

**Usage:**
- Type `/squad` in the chat input box
- Same as `/all-agents` - all 4 subagents launch simultaneously
- Use with a task description for coordinated work: `/squad implement user authentication`

**What it does:**
Spawns all 4 agents in parallel:
- `code-reviewer` - Reviews code quality and security
- `frontend-specialist` - Handles frontend implementation
- `backend-specialist` - Handles backend/API implementation
- `test-runner` - Runs tests and provides feedback

**Example Usage:**
```
/squad review the authentication system for security issues
/squad implement a new contact form with validation
/squad refactor the API endpoints to use consistent error handling
```

**See also:** `/all-agents` for detailed documentation
